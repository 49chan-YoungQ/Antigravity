# Day 0: 내 첫 번째 아이디어 웹 프로젝트 요약

지금까지 진행된 **Day 0** 프로젝트의 주요 작업 내용과 진행 이력을 정리한 문서입니다.

---

## 📌 1. 프로젝트 목표
- **"화면 중앙에 내 첫번째 아이디어"**라는 문장을 사용자 화면 정중앙에 수려하고 역동적으로 표현하는 웹페이지를 구축합니다.
- 단순한 텍스트 표시를 넘어 프리미엄 디자인 트렌드(글래스모피즘, 3D 반응형 모션, 라이트/다크 테마, 캔버스 이펙트)를 반영합니다.

## 🎨 3. 주요 구현 기능
- **글래스모피즘(Glassmorphism) 카드**: 배경 블러와 그라데이션 광원을 조화시켜 반투명한 유리 질감의 카드를 구현했습니다.
- **3D Tilt 효과**: 마우스 포인터의 위치에 반응하여 카드가 실시간으로 부드럽게 기울어지는 몰입감 있는 효과를 제공합니다.
- **인터랙티브 파티클 캔버스**: 
  - 화면 전반에 은은하게 떠다니는 배경 파티클 탑재.
  - 마우스 클릭 또는 전용 버튼 클릭 시 색동 불꽃이 터져 나오는 Sparkle Burst 효과 적용.
- **다크/라이트 테마**: 사용자 설정에 맞게 눈이 편안한 다크 모드와 화사한 라이트 모드를 자유롭게 전환 가능합니다.
- **문장 복사 기능**: 버튼 클릭 시 텍스트가 클립보드에 복사되고 이와 동시에 자연스럽게 페이드인/아웃되는 토스트 팝업 알림이 작동합니다.

## 🚀 4. 배포 및 저장소 연동 이력

### A. Github 원격 저장소 연동
- **주소**: [https://github.com/49chan-YoungQ/Antigravity](https://github.com/49chan-YoungQ/Antigravity)
- **이력**:
  1. 로컬 환경에 Git 저장소 초기화.
  2. Github CLI 인증 절차(OAuth 일회용 코드 사용)를 거쳐 계정 연동 및 퍼블릭 저장소 `Antigravity` 자동 생성.
  3. `Day_0` 내부 리소스를 `main` 브랜치에 성공적으로 푸시 완료.

### B. Vercel 배포 가이드 (404 오류 해결)
파일을 하위 서브 폴더(`Day_0/idea_presentation`)로 이동했기 때문에 Vercel 배포 시 `404: NOT_FOUND`가 발생합니다. 아래 설정 중 하나로 해결할 수 있습니다.

1. **Root Directory 설정 변경 (권장)**:
   - Vercel 대시보드 진입 ➡️ **Settings** ➡️ **General**로 이동.
   - **Root Directory** 경로를 **`Day_0/idea_presentation`** 으로 수정 후 저장 및 Redeploy.
2. **다이렉트 URL 주소 접속**:
   - 배포된 URL 뒤에 직접 하위 경로를 추가하여 접속:
     `https://<your-vercel-domain>/Day_0/idea_presentation/index.html`
