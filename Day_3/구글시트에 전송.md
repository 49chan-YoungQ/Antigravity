# 구글 시트에 실시간 투표 데이터 전송 (POST) 구현 요약

## 1. 개요 및 베스트 프랙티스 도입
기존 애플리케이션은 공개된 구글 시트의 CSV 주소를 통해 데이터를 **읽어오는(GET)** 기능만 구현되어 있었습니다. 사용자가 투표를 했을 때 이를 구글 시트에 실시간으로 **기록(POST)** 하기 위한 작업을 진행했습니다.

정적 웹사이트(HTML/JS) 환경에서 구글 API 키나 인증 정보를 프론트엔드 코드에 직접 포함할 경우, **보안(인증 정보 노출) 문제**가 발생합니다. 이를 방지하고 안전하게 데이터를 기록하기 위해 Vercel 등 정적 호스팅 환경에서 권장되는 베스트 프랙티스인 **Google Apps Script(GAS)를 미들웨어(백엔드 API)로 활용**하는 구조를 채택했습니다.

---

## 2. 프론트엔드(`index.html`) 변경 사항

1. **`GOOGLE_APPS_SCRIPT_URL` 상수 추가**
   - 구글 Apps Script에서 발급받을 웹 앱 URL을 저장할 변수를 선언했습니다.

2. **`castVote()` 함수 비동기화 및 API 연동**
   - 사용자가 메뉴를 선택하고 투표 버튼을 누를 때 호출되는 함수를 `async` 함수로 변경했습니다.
   - `fetch()` API를 사용하여 `GOOGLE_APPS_SCRIPT_URL` 주소로 폼 데이터(`application/x-www-form-urlencoded`) 형식의 **POST 요청**을 전송하는 로직을 추가했습니다.
   - 구글 시트로 성공적으로 데이터가 전송되면 토스트 알림을 띄우고 카운트를 즉시 증가시켜 사용자에게 빠른 피드백을 제공합니다.

3. **UX 개선 (로딩 처리)**
   - 투표 데이터가 네트워크를 통해 전송되는 동안 투표 버튼이 `투표 전송 중...`이라는 텍스트와 회전하는 스피너 아이콘으로 변경되며 버튼이 비활성화됩니다.
   - 이를 통해 통신 지연 중 발생할 수 있는 중복 투표 클릭을 방지했습니다.

---

## 3. 구글 시트 백엔드 (Apps Script) 설정 방법

프론트엔드에서 보낸 투표 데이터를 시트에 기록하려면 구글 시트 측에 다음과 같은 웹 앱을 배포해야 합니다.

### 설정 순서
1. 투표 결과를 저장할 구글 시트를 열고, 상단 메뉴에서 **[확장 프로그램] > [Apps Script]** 를 클릭합니다.
2. 아래의 코드를 붙여넣습니다:
   ```javascript
   function doPost(e) {
     const headers = {
       "Access-Control-Allow-Origin": "*",
       "Content-Type": "application/json"
     };
     
     try {
       const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
       const timestamp = new Date();
       const menu = e.parameter.menu; 
       
       // 시간과 메뉴 이름을 시트에 기록
       sheet.appendRow([timestamp, menu]);
       
       return ContentService.createTextOutput(JSON.stringify({ "status": "success" }))
         .setMimeType(ContentService.MimeType.JSON);
         
     } catch (error) {
       return ContentService.createTextOutput(JSON.stringify({ "status": "error", "message": error.toString() }))
         .setMimeType(ContentService.MimeType.JSON);
     }
   }
   ```
3. 에디터 우측 상단의 **[배포] > [새 배포]**를 클릭하여 **[웹 앱]** 유형으로 설정합니다.
   - 엑세스 권한: **모든 사용자 (Anyone)**
4. 배포 후 생성된 **웹 앱 URL**을 복사하여 `index.html`의 `GOOGLE_APPS_SCRIPT_URL` 값으로 지정해 주면 모든 연동이 완료됩니다.
