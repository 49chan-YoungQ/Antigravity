# Project Day 3: 실시간 점심 메뉴 투표 서비스

본 프로젝트는 `Day_2`까지 쌓아온 프리미엄 웹 컴포넌트 설계 역량과 동적 캔버스 물리 이펙트를 한 단계 발전시켜, 실시간 데이터 갱신과 모형 시뮬레이션을 탑재한 **실시간 점심 메뉴 투표 웹 서비스**입니다.

---

## 📅 주요 구현 이력 및 개발 노트

### 1. 실시간 데이터 동기화 및 2컬럼 레이아웃 재구성
- **기존 상태**: 세로 방향의 단일 흐름 카드 구조였습니다.
- **수행 작업**:
  - 데스크톱 화면 넓이를 활용할 수 있도록 좌우 **2컬럼 그리드 레이아웃**을 구축했습니다.
  - **좌측**: 투표할 메뉴를 선택하는 2x2 반응형 카드 영역과 최종 제출용 "투표하기" 버튼을 배치했습니다.
  - **우측**: 실시간 누적 투표수 계기판, 게이지 차트, 시뮬레이션 제어 도구, 초기화 버튼을 유기적으로 결합했습니다.
  - 화면 폭이 좁아지는 모바일 환경에서는 1열 세로 배열로 흐름이 매끄럽게 전환되도록 반응형 미디어 쿼리를 전면 재조정했습니다.

### 2. 프리미엄 '오렌지/골드 & 미드나잇 블루' 테마 리브랜딩
- **수행 작업**:
  - 점심 메뉴 투표라는 컨셉에 맞추어 식욕을 돋우는 따뜻한 **오렌지/골드** 톤을 메인 포인트 컬러로 활용했습니다.
  - **배경 파티클**: 연녹색 파티클을 오렌지/앰버/로즈 톤의 따뜻한 불꽃 아지랑이 파티클로 튜닝하여, 캔버스 밑바닥에서 아지랑이가 천천히 솟아올라 흐르는 신비로운 분위기를 극대화했습니다.
  - 다크 모드 활성화 시 미드나잇 블랙 카본 배경과 어우러져 찬란하게 타오르는 화염 에센스 같은 대비 효과를 제공합니다.

### 3. 실시간 투표 시뮬레이션 시스템 도입
- **수행 작업**:
  - 백엔드 서버 없이도 실시간으로 그래프가 채워지고 변동되는 재미 요소를 주기 위해 **가상 투표 시뮬레이션 엔진**을 코딩했습니다.
  - 무작위 이름 조합 기능(예: '배고픈 길동이', '식사중인 부장님')을 활용해 주기적으로 투표를 자동으로 발생시키며, 이에 맞추어 실시간 현황 차트가 동적으로 움직입니다.
  - 속도(주기) 제어 장치(느림 7초, 보통 3초, 빠름 1.2초)를 제공하여 사용자가 차트 게이지 채워지는 속도를 직관적으로 바꿀 수 있습니다.

### 4. 마이크로 인터랙션 및 특수효과 강화
- **수행 작업**:
  - **메뉴 카드**: 선택 시 테두리가 빛나는 오렌지 글로우 효과와 함께 우측 상단 체크 뱃지가 튀어나오며, 마우스 호버 시 3D Tilt 스케일링이 들어가 터치감을 높였습니다.
  - **Sparkle Burst (불꽃 이펙트)**: 사용자가 투표를 던지거나 시뮬레이터가 투표를 기록할 때, 캔버스 전역과 버튼 뒤편에서 반짝이는 다이아몬드 불꽃들이 탄성 물리에 맞추어 터지며 중력으로 인해 떨어지는 극강의 시각 피드백을 연출했습니다.
