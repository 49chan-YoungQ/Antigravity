# Project Day 2: 나의 첫 서비스 (3구역 레이아웃 및 테마 리브랜딩)

본 프로젝트는 `Day_1`에서 완성한 인터랙티브 응원 페이지를 기반으로, 실제 웹 서비스 수준의 레이아웃 구조화 및 브랜딩 컬러 스킴을 적용하여 완성도를 크게 높인 웹 페이지입니다. 

---

## 📅 주요 구현 이력 및 개발 노트

### 1. 3구역(헤더/본문/푸터) 구조 분할 및 레이아웃 구조화
- **기존 상태**: 화면 중앙에 1개의 카드가 떠 있는 단조로운 구조였습니다.
- **수행 작업**:
  - `body`를 세로 방향 플랙스 박스(`flex-direction: column; min-height: 100vh;`)로 재구조화했습니다.
  - 최상단에 `<header>`, 중간 본문에 `<main class="app-main">`, 최하단에 `<footer class="app-footer">`를 배치하여 화면 전체 영역을 짜임새 있게 삼등분했습니다.
  - 푸터 텍스트로 `"© 2026 49chan Service"`를 적용했습니다.

### 2. 헤더 상단 바화 및 네비게이션 메뉴 설계
- **수행 작업**:
  - 기존의 중앙 정렬 헤더에서 `display: flex; justify-content: space-between; align-items: center;`를 적용한 가로 상단 바 형태로 개선했습니다.
  - 타이틀 `"나의 첫 서비스"`는 좌측에 정렬하고, 우측에는 `<nav>` 태그로 `"소개"`, `"기능"`, `"메뉴"` 링크를 가로로 배치했습니다.
  - 각 링크 사이에 `24px`의 간격을 주어 가독성을 높였습니다.
  - 마우스를 올릴 때 흰색 활성화와 함께 위로 부드럽게 1px 올라가는 호버 애니메이션 인터랙션을 적용했습니다.
  - 모바일(600px 이하) 환경에 맞추어 상단 바 여백과 메뉴 간격이 자동으로 줄어드는 반응형 미디어 쿼리를 설계했습니다.

### 3. 본문 이중 섹션화 및 사용자 후기 컴포넌트 추가
- **수행 작업**:
  - 본문 영역을 상하로 나뉜 두 개의 독립된 글래스모피즘 카드로 나누었습니다.
  - **섹션 1 (핵심 기능 소개)**: 기존의 "오늘도 힘내세요" 문장 출력부와 문장 복사, 반짝임 효과 버튼을 담았습니다.
  - **섹션 2 (사용자 후기)**: 가상의 후기 2개를 담은 고급스러운 피드백 영역을 추가했습니다.
  - **프로필 사진 아바타 생성**: AI 생성형 이미지 툴을 활용하여 고품질의 아바타 2종(`user_profile_1.png`, `user_profile_2.png`)을 생성하고 `images/` 하위에 배치하여 연동했습니다.
  - **마이크로 애니메이션**: 후기 카드에 호버 시 카드가 우측 위로 슬라이드(`translateX(6px) translateY(-2px)`)하고, 프로필 아바타가 회전하면서 커지는 입체적인 액션을 적용했습니다.

### 4. 연녹색 & 숲속 다크 그린 테마 리브랜딩
- **수행 작업**:
  - **주 색상**: 상쾌한 느낌의 **연한 녹색(Light Green)**으로 변경했습니다. 라이트 모드에서는 선명한 그린 계열 그라데이션, 다크 모드에서는 눈에 부드러운 밝은 연녹색 그라데이션을 적용했습니다.
  - **보조 색상**: 주 색상과 대비를 이루면서 고급스러움을 더해주는 **어두운 숲속 색상(Forest Dark Accent)** 계열을 정의했습니다. 상단 바 헤더의 배경으로 적용되었습니다.
  - **버튼 컬러스킴**: 기본 상태에서는 주 색상(연녹색)을 띠고, 마우스를 올리면 보조 색상(진녹색)으로 부드럽게 그라데이션 전환되는 인터랙션을 구현했습니다.
  - **기타 요소 매칭**: 뒷배경을 흐르는 은은한 파티클들의 색상도 에메랄드/연녹색 조합으로 튜닝하여 전체적인 브랜딩 일관성을 지켰습니다.

## 🛠️ 기술 스택 및 구조 정의
- **Core**: HTML5, Vanilla JavaScript, CSS3
- **아이콘**: FontAwesome 6.4.0 (CDN)
- **폰트**: Pretendard (Korean Sans-Serif CDN)
- **디자인 스타일**: Glassmorphism, Responsive Grid/Flex layout, CSS Custom Properties (Variables)
- **디렉토리 구조**:
  ```bash
  Day_2/
  ├── index.html         # 메인 마크업 및 스타일, 스크립트 파일
  ├── README.md          # 프로젝트 구현 역사 및 기능 명세서
  └── images/            # 생성형 AI 기반 프로필 이미지 폴더
      ├── user_profile_1.png  # 김민수 개발자 아바타 이미지
      └── user_profile_2.png  # 이지원 디자이너 아바타 이미지
  ```
