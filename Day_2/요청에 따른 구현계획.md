# 구현 계획: Day_2 3구역(헤더/본문/푸터) 레이아웃 적용

`Day_1/index.html` 파일을 `Day_2` 폴더로 복사하고, 전체 화면을 **헤더**, **본문**, **푸터**의 3구역으로 나누어 구조화하며 요구사항에 맞는 스타일링을 적용합니다.

## 제안하는 변경 사항

### 1. `Day_2/index.html` 신규 생성
`Day_1/index.html`을 기반으로 아래의 변경 사항을 적용하여 [Day_2/index.html](file:///d:/12. Antigravity/Day_2/index.html)을 생성합니다.

- **레이아웃 구조 개편**:
  - `body`를 세로 방향 플랙스 박스(`flex-direction: column`)로 변경하여 헤더, 본문, 푸터가 수직으로 나란히 배치되게 하고, 높이가 100vh 전체를 채우도록 합니다.
  - 스크롤 가능하도록 `overflow: hidden;`을 `overflow-y: auto;`로 보완합니다.

- **헤더 영역 추가 (`<header>`)**:
  - 제목 `"나의 첫 서비스"`를 포함하는 헤더 영역을 최상단에 배치합니다.
  - 헤더 배경으로 아름다운 파란색 그라데이션(`linear-gradient(135deg, #2563eb, #1d4ed8)`)을 적용합니다.

- **본문 영역 (`<main>`)**:
  - 기존의 문장과 카드 컴포넌트(버튼들 포함)를 본문 중앙에 배치합니다.
  - `main` 태그에 `flex: 1`을 주어 헤더와 푸터를 제외한 화면 전체 영역을 차지하고, 내부 컨텐츠를 중앙에 배치(`justify-content: center`, `align-items: center`)하도록 구현합니다.

- **푸터 영역 (`<footer>`)**:
  - 푸터 텍스트 `"© 2026 49chan Service"`를 반영합니다.
  - 하단에 고정된 회색 배경 디자인을 적용합니다. 라이트 모드에서는 연한 회색(`--footer-bg: #f8fafc`), 다크 모드에서는 어두운 슬레이트 그레이 계열(`--footer-bg: #1e293b`)로 자연스럽게 색상이 연동되도록 CSS 변수를 활용합니다.

- **기타 변경 사항**:
  - SEO를 위해 타이틀 태그 내용을 `나의 첫 서비스`로 변경합니다.
  - 마우스 무브 카드 틸트 이펙트 및 파티클 캔버스 등의 동적 디자인 요소가 변경된 구조에서도 완벽하게 작동하도록 보완합니다.

## 검증 계획

### 수동 검증
- 작성된 페이지를 브라우저를 통해 실행합니다.
- 화면 상단에 파란색 배경의 **"나의 첫 서비스"** 헤더가 올바르게 렌더링되는지 확인합니다.
- 화면 중앙에 기존 문장("오늘도 힘내세요")과 버튼들이 깔끔하게 세로/가로 정렬되는지 확인합니다.
- 화면 하단에 회색 배경과 함께 **"© 2026 49chan Service"**가 노출되는지 확인합니다.
- 테마 변경(다크 모드 / 라이트 모드) 시 푸터의 회색 배경이 테마와 어우러지게 색상이 변경되는지 검증합니다.
- 브라우저 반응형 레이아웃 및 마우스 무브 카드 틸트 이펙트가 부드럽게 작동하는지 확인합니다.
