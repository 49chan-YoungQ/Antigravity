## 🎨 2. 주요 구현 및 업그레이드 기능

### A. 디자인 시스템 리뉴얼
* **배경 그라데이션**: 
  * 기본 배경을 세련되고 밝은 실버 느낌의 **플래티넘 회색 그라데이션**(`linear-gradient(135deg, #f8fafc 0%, #e2e8f0 100%)`)으로 업그레이드했습니다.
  * 다크 모드 전환 시에는 이에 대응하는 **깊이감 있는 딥 슬레이트 그레이 그라데이션**으로 부드럽게 바뀝니다.
* **글래스모피즘(Glassmorphism) 카드 고도화**:
  * 밝은 배경 위에서도 유리 질감이 돋보이도록 카드 상단 안쪽에 **화이트 하이라이팅 라인**(`inset 0 1px 0 rgba(255, 255, 255, 0.8)`)을 추가해 현실적인 유리 패널의 두께감을 살렸습니다.
  * 백그라운드에 은은하게 유영하는 **진주빛 블루(Pearl Blue)**와 **오렌지 피치(Orange Peach)** 소프트 조명 구체를 배치하여 다채로운 굴절 효과를 더했습니다.
* **타이포그래피 및 가독성 최적화**:
  * 메인 문장의 폰트 크기를 **`40px`**(모바일 기기에서는 자동으로 **`32px`** 축소)로 최적화했습니다.
  * 테마 전환 시 글자가 묻히지 않도록, 다크 모드에서는 **맑은 스노우 화이트 네온 글로우 그라데이션**으로, 라이트 모드에서는 **짙은 차콜 슬레이트 그라데이션**으로 글자 색상이 유기적으로 바뀝니다.

### B. 동적 문장 슬라이더 ("다음 문장 보기" 버튼)
* **옐로우 그라데이션 버튼**: 카드 본문 하단에 호버/액티브 인터랙션이 연동된 밝고 선명한 옐로우 버튼을 추가했습니다.
* **위로 문장 배열 (Sentences Pool)**: 클릭 시 5가지의 격려와 위로의 문장이 순환하여 나타납니다.
  1. *오늘도 힘내세요*
  2. *당신은 존재만으로 빛나요*
  3. *포기하지 않는 마음*
  4. *오늘 하루도 수고 많았어요*
  5. *늘 당신을 응원합니다*
* **페이드 트랜지션 애니메이션**: 문장 변경 시 텍스트가 위아래로 부드럽게 흩어지며 투명해지는 Fade Out/In 효과를 통해 자연스러운 화면 전환을 이끌어냈습니다.

### C. 활성 문장 매핑 동적 버튼 색상
사용자가 감상하고 있는 문구의 정서에 맞춰 버튼의 색상이 실시간으로 자동 변경되어 몰입감을 높입니다.
* **"당신은 존재만으로 빛나요"** & **"늘 당신을 응원합니다"** ➡️ **`파란색(Blue)`**
* **"포기하지 않는 마음"** ➡️ **`적색(Red)`**
* **"오늘 하루도 수고 많았어요"** ➡️ **`회색(Gray)`**
* **"오늘도 힘내세요"** (기본) ➡️ **`노란색(Yellow)`**

### D. 고해상도 폭죽 물리 엔진 (Firework Explosion Particles)
* **DPI 리스케일링 (High-DPI Scaling)**:
  * 4K, Retina 등 고해상도 디스플레이 환경에서 캔버스 그래픽이 번지거나 흐릿하게 보이는 현상을 근본적으로 차단하기 위해 `devicePixelRatio`를 추적하여 물리 버퍼를 스케일링했습니다.
* **4-Pointed 크로스 스타 그래픽**:
  * 터져 나가는 입자들을 **4방향 십자별(Diamond Star)**로 드로잉하고, 중심부에 백색 **광학 렌즈 플레어 크로스 라인**과 핫 센터를 입혀 광채의 선명도를 높였습니다.
* **연기 잔상 효과 (Spark Trails)**:
  * 불꽃이 비행한 자리에 은은하고 투명한 **트레일 라인(Trail Line)**을 렌더링하여 폭죽 꼬리가 남는 시각적 쾌감을 극대화했습니다.
* **중력(Gravity) 및 대기 마찰(Friction) 물리 통합**:
  * 초기 폭발 후 공기 저항으로 인해 속도가 감속하며, 중력 가속도(`0.12`)를 받아 포물선을 그리며 **아래로 낙하하고 소멸하는 실제 불꽃놀이의 물리 동역학**을 그대로 시뮬레이션했습니다.
* **섬광 깜빡임 (Flickering) & 혼합 모드 (Screen Blend)**:
  * 입자가 완전히 사라지기 직전에 깜빡거리며 부서지는 스파클 섬광 효과와 겹치는 입자끼리 강렬히 융합해 밝아지는 스크린 블렌딩 모드를 추가했습니다.
* **배경 트윙클 별무리**:
  * 화면 전체에 은하수처럼 깔려있는 배경 별자리들이 부드럽게 숨 쉬듯 깜빡거리도록(Twinkling) 처리하여 대기 상태에서의 완성도를 더했습니다.

## 🛠️ 3. 크로스 브라우저 호환성 및 표준화 (Warning 해결)
* CSS 린터(Linter)에서 제시한 `-webkit-mask` 전용 접두사 사용 경고를 조치하기 위해 표준 `mask` 선언을 명시해 브라우저 호환성 표준을 완벽히 만족시켰습니다.
```css
-webkit-mask: linear-gradient(#fff 0 0) content-box, linear-gradient(#fff 0 0);
mask: linear-gradient(#fff 0 0) content-box, linear-gradient(#fff 0 0);
```
